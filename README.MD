# 한국 응급실/병원 지도 서비스 (Streamlit)

이 프로젝트는 CSV 파일에 포함된 병원/응급실 정보를 기반으로
지도에서 위치를 확인하고, 지역/요일/병원명 검색 등의 필터 기능을 제공하며,
필터 결과가 없을 경우 가장 가까운 응급실을 안내하는 웹 애플리케이션입니다.

## 🚀 주요 기능

- 병원 위치를 지도(PyDeck)로 시각화
- 지역, 요일, 시간 텍스트 검색, 병원명 검색 필터 제공
- 필터에 해당 병원이 없을 경우 사용자 위치 기반 최단거리 응급실 추천
- 한국어 UI 지원

## 📁 CSV 파일 형식 예시

필수 컬럼명:

| 컬럼 | 설명 |
|------|------|
| 이름 | 병원명 |
| 주소 | 지역/주소 |
| 위도 | float (위도) |
| 경도 | float (경도) |

선택(있으면 필터에 자동 반영):

| 컬럼 | 설명 |
| 요일별 운영 여부 | 월, 화, 수, 목, 금, 토, 일, 공휴일 |
| 응급실 | 응급 여부 정보 |

예시 CSV는 `hospitals.csv` 파일을 참고하세요.
